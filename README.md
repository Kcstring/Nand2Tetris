这是我关于Nand2Tetris项目的具体实现，

项目官网：https://www.nand2tetris.org/

整个项目对我的帮助十分之大，囊括了从数字电路，体系结构，编译原理以及操作系统的知识，尽管为了简化课程难度，课程做了许多简化处理，但它给初学者带来的震撼与收获并没有降低，假如没有它，我也许至今也不会明白本科中我们那段枯燥无味的数字电路课程意义何在，尽管项目中只涉及到了一小部分（逻辑门）

其中：

第一部分：大部分在https://nand2tetris.github.io/web-ide/chip/ 网站上完成，故基本没有保留源码；

第二部分：基本上完成了，但由于时间问题，关于OS和编译器一小部分引用了其他博主的解答。

项目整体难度不高，但全部完成还是需要花费一段时间，感兴趣的请自行去官网查看
